# Personal Finance Tracker

**Personal Finance Tracker** — мобильное приложение для Android, предназначенное для учёта личных доходов и расходов. Позволяет регистрировать транзакции, настраивать повторяющиеся платежи, просматривать статистику в виде круговой диаграммы и получать уведомления о предстоящих платежах.

Приложение разработано в рамках курсовой работы с использованием современных технологий Android-разработки: **Kotlin**, **Jetpack Compose**, **Room**, **MVVM**, **MPAndroidChart**.

---

## Возможности

- Регистрация и вход пользователя (с хэшированием пароля SHA-256)
- Добавление, редактирование и удаление транзакций (доходы/расходы)
- Категории транзакций: Зарплата, Еда, Транспорт, Развлечения и др.
- Повторяющиеся платежи с настраиваемой периодичностью (ежедневно, еженедельно, ежемесячно, ежеквартально)
- Уведомления о предстоящих платежах (через `AlarmManager`)
- Визуализация статистики в виде круговой диаграммы (доходы vs расходы)
- Фильтрация статистики по периоду: неделя, месяц
- Экран FAQ с ответами на частые вопросы
- Адаптивный интерфейс под разные размеры экранов

## Технологии

| Технология             | Назначение |
|------------------------|----------|
| **Kotlin**             | Основной язык разработки |
| **Jetpack Compose**    | Современный UI-интерфейс |

## Установка и запуск

### Требования
- Android 8.0 (API 26) и выше
- Установленный **Android Studio** (рекомендуется Koala или новее)

---

### Шаги по установке

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/RascallySine18/personal-finance-tracker.git
2. Откройте проект в Android Studio
3. Дождитесь синхронизации Gradle
4. Запустите приложение на эмуляторе или подключённом устройстве:
- Выберите устройство (API 26+)
- Нажмите Run
